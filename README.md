# Hyprland Dotfiles

My Hyprland rice setup on Arch Linux with Waybar, Rofi, Kitty, and Dunst.

## Components

- **Window Manager**: Hyprland 0.55.4
- **Bar**: Waybar (transparent, rounded modules)
- **Launcher**: Rofi (adi1090x theme, type-3 style-7)
- **Terminal**: Kitty (Catppuccin Mocha)
- **Notifications**: Dunst
- **Wallpapers**: swaybg (13 anime wallpapers)

## Keybinds

| Key | Action |
|-----|--------|
| `Super + Return` | Open terminal |
| `Super + Q` | Kill active window |
| `Super + Space` | App launcher (Rofi) |
| `Super + W` | Cycle wallpaper |
| `Super + 1-9` | Switch workspace |
| `Super + Shift + 1-9` | Move window to workspace |
| `Alt + Return` | Fullscreen |
| `Alt + F4` | Kill window |

## Install

```bash
git clone https://github.com/hehegriseo/dotfiles.git ~/.config
```

## Wallpaper

Drop images in `~/Pictures/wallpapers/` and use `Super+W` to cycle.
